guise.util.google_search_html(q, num=10, start=0, lr=None, on_error=<function print_status_code_and_content>, **kwargs)[source]

Get’s the html of google results (start through start+num) for query q.

Parameters:

  • q – query

  • num – the number of results you want (default is 10 – docs say it’s also the max, but haven’t seen that to be true)

  • start – Which result to start with (play with start and num to page through results

  • lr – Language restriction (e.g. “lang_en” for English)

  • on_error – if given, func that is called on response if status code not 200

  • kwargs – Extra params:

See https://developers.google.com/custom-search/v1/reference/rest/v1/cse/list for extra parameters and descriptions.
