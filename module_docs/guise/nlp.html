class guise.nlp.TFIDFCalculator(reference_corpus, idf_type='frequency', stopwords=None, custom_idf=None)[source]

A TF-IDF calculator that computes TF-IDF scores for documents based on a corpus.

Parameters:

  • reference_corpus (pandas.Series) – The corpus word counts to use as reference for IDF calculations. The corpus_counts will be taken, or computed from reference_corpus.

  • idf_type (str, optional) – The type of IDF calculation to use. Options are as follows (where df is the ‘document frequency’ of a word (a word count) and N is the total number of documents in the corpus): - ‘standard’: IDF = log(N / (df)) - ‘smooth’: IDF = log(1 + N / (df)) - ‘probabilistic’: IDF = log((N - df) / df) - ‘frequency’: IDF = log((total_counts_corpus + 1) / (corpus_counts + 1)) Default is ‘frequency’.

  • stopwords (set or list, optional) – A set or list of stop words to exclude from calculations.

  • custom_idf (dict or pandas.Series, optional) – A mapping from words to custom IDF values.

idf_

Inverse Document Frequency (IDF) values computed from the corpus.

Type:

pandas.Series

default_idf_

Default IDF value for words not found in the corpus.

Type:

float

Examples

>>> import pandas as pd
>>> # Sample corpus word counts
>>> reference_corpus = pd.Series({
...     'word1': 300,
...     'word2': 200,
...     'word4': 500,
...     'trump': 100
... })
>>> stopwords = {'the', 'and', 'is'}
>>> custom_idf = {'trump': 5.0}
>>> tfidf_calculator = TFIDFCalculator(reference_corpus,
...                                    idf_type='standard',
...                                    stopwords=stopwords,
...                                    custom_idf=custom_idf)
>>> # Sample document word counts
>>> doc_counts = pd.Series({
...     'word1': 3,
...     'word2': 2,
...     'trump': 4,
...     'harris': 2,
...     'the': 10
... })
>>> tfidf_scores = tfidf_calculator(doc_counts)
>>> print(tfidf_scores)
word1    -1.17...
word2    -0.71...
trump     1.81...
harris    0.90...

guise.nlp.stem_based_word_mapping(words: Iterable[str], stemmer: Callable | str | None = 'lancaster') → Mapping[str, str][source]

Returns a dict of {word: replace_with_word, …} mapping based on stemming.

stemming is great to reduce the number of words by replacing several

The words (or word counts) are stemmed, then for each stem, the word with the highest count gets to represent the others.

>>> _words = ['happy', 'happier', 'happiest', 'happily', 'sad', 'sadly']
>>> word_count = Counter(_words)
>>> stem_based_word_mapping(word_count)
{'happy': 'happy',
 'happier': 'happy',
 'happiest': 'happiest',
 'happily': 'happy',
 'sad': 'sad',
 'sadly': 'sad'}
